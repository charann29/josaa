# 10 — Product Vision (SaaS + Service)

> Captured verbatim-in-spirit from Charan during this session. This is the seed for the next chat's business-proposal work.

## Core idea
**Not just a simple AI to help with the counselling part — a *service business* built around it.** The AI advisor is the engine; the offering is an end-to-end admissions-counselling service.

## Scope ambition
**Not only the aligned JoSAA system (IITs / NITs / IIITs / GFTIs)** — also **state-board counselling** (the many state-level engineering/college admission processes, e.g. state CETs). i.e. one platform that handles *both* central (JoSAA) and *state* counselling.

## What the engine already does (proven this session — see other notes)
This whole session is a working prototype of the value prop, for one student:
- Ingests official **closing-rank history** (multi-year) → computes **per-student reachability** (SAFE/LIKELY/BORDERLINE) against the student's exact rank + category + quota + PwD pools. → [[04 — Reachability Methodology]]
- Builds a **transparent points score** (branch + career + NIRF + brand + culture + user preference) and a **percentile-ranked, priority-ordered choice list**. → [[05 — Scoring System (Points)]]
- Applies **personal filters**: disability **restriction safety**, geography/weather, brand tiers, branch-suitability. → [[06 — Filters & Rules]]
- **Simulates the likely allotment** from historical cutoffs and **rates the list**. → [[07 — Final List & Predicted Outcome]]
- **Automates portal entry** with restriction guards + human-in-the-loop lock. → [[08 — Automation Engine]]

## Why it's differentiated
- **Reachability + restriction-awareness per individual** (esp. PwD/medical restrictions) — most counsellors don't model this rigorously.
- **Explainable scoring** (every seat shows why it ranks where it does) → builds trust, justifies fees.
- **Outcome simulation** against real multi-year data, not vibes.
- Bridges **central + state** counselling in one place.

## Possible model layers (to develop next chat)
- **Self-serve SaaS:** student enters rank/category/prefs → gets the scored list + simulation.
- **Assisted service:** human counsellor + AI co-pilot; premium, high-touch (incl. PwD/edge cases).
- **B2B2C:** white-label for coaching institutes / schools / state bodies.
- **Data moat:** curated, cleaned multi-year cutoff data across central + every state board.
