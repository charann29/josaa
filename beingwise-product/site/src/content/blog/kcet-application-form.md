The KCET Application Form for 2025 is now open. The registration link is active from **23rd Jan 2025 to 24th Feb 2025**. The last date to apply for the KCET Exam will be **24th Feb 2025**. The **Admit card** will be released on **March 25th, 2025**.

The KCET exam will start on 16th April 2025. For Physics and Chemistry Papers the exam is going to be conducted on **16th April 2025** and for Maths Papers the exam is going to be conducted on **17th April 2025**. The **Kannada Language Test** is going to be conducted on **April 15th, 2025**.

## Steps to Fill out the Application Form for KCET

**Step 1:** Log in to the [Official Website](https://cetonline.karnataka.gov.in/kea/indexnew)

**Step 2:** Access the [Online UGCET Registration Link](https://cetonline.karnataka.gov.in/onlineapplication2025/forms/appchecklist.aspx)

**Step 3:** Keep the Following **documents required for the KCET application** ready before going to register for the KCET examination

- 10th Marks Card for entering the Correct Registration Number.
- Have the Caste Certificate / Caste Income Certificate / 371(j) certificate, NCLC to enter the RD Number.
- Have scanned images of the latest Photograph and signature in JPG format according to the Specifications.

|  |  |  |  |
| --- | --- | --- | --- |
| **Document** | **Specifications** | **Dimensions** | **File Format** |
| **Photograph of the Candidate** | The photograph should only be on a white background.  Passport Size photo is only allowed mobile photo is not allowed.  The Photograph should be clear | 3.5c.mX4.5c.m | 5 to 40KB |
| **Signature of the Candidate** | The candidate’s signature should be on a white A4 sheet with a black ballpoint pen.  Scan and crop the signature photo so that only the signature is visible. | 3.5c.mX1.5c.m | 5 to 40 KB |

**Step 4:** Click On Proceed to Registration for new users.

**Step 5:** The candidate will now be able to access the registration form, they have to enter the following details

- Mobile Number
- Create Password
- Upload the photo and submit it.
- OTP will be sent to the candidate’s registered mobile number. Hence, the candidate has to provide their mobile number only in this section.

**Step 6:** Upon successful registration, candidates will receive an Application Number and User-ID to their registered mobile phone number. The Application Number and user ID will also be displayed on the screen.

**Step 7:** After Registration Click on “Proceed to Login for Registered Users”.Login with the following details

- USER ID
- Password
- OTP or by Face Recognition

**Step 8:** The details should be completed tab-wise one after the other. The tabs that are involved here are:

- General Details
- Eligibility Reservation
- Study Details
- NRI Ward
- Course Details

**Step 9:** Candidates are advised to take a draft printout of the application form and check all the details including reservation, RD Number, Medium and Area, and Special Category(If applicable). If there is any error log in with OTP or facial recognition and change the details.

**Step 10:**Click on Proceed to Payment if there are no changes

**Step 11:**Login using the User ID and Password, select the payment gateway and pay the fees.

**Step 12:**Take the Final Printout along with the Claim Certificate.

## Frequently Asked Questions

### **1. When is the KCET Application Form 2025 Last date?**

The last date for applying for the KCET Exam Online is 24th Feb 2025.

### **2.  What Happens if there are any errors in a candidate’s Application?**

If there are any errors in the Candidates application the application correction window facility will be available to correct the error.

### **3. Does  KCET Have any preparatory exams?**

Yes, The KEA exam conducting Authority for KCET will now have preparatory exams so that the candidates will become familiar with the OMR sheets and filling of OMR Sheets.

### 4. What if I fail to Complete the Steps Given In Registration?

If a candidate fails to complete the steps in the registration process then their application will be rejected.

### 5. When is the Math Paper Conducted for KCET?

The Math Paper will be conducted for KCET on April 17th,2025.
