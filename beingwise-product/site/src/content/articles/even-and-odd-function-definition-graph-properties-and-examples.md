# Even and Odd Function: Definition, Graph, Properties and Examples

In mathematics, functions are classified in various ways, and understanding whether a function is even or odd is fundamental for many advanced topics. This concept is particularly important for students preparing for competitive exams like JEE Main, as it's a recurring theme in the Class 11 Mathematics curriculum.

## Key takeaways

*   Even and odd functions are defined by their symmetry properties when the input variable is negated.
*   An even function remains unchanged when `x` is replaced by `-x`, meaning `f(-x) = f(x)`.
*   An odd function changes its sign when `x` is replaced by `-x`, meaning `f(-x) = -f(x)`.
*   These classifications are crucial for areas like algebra, calculus, and Fourier analysis.

## What are Even and Odd Functions?

Even and odd functions are types of functions categorized by how they behave when their input is reversed (i.e., `x` becomes `-x`). This behavior reveals specific symmetries in their graphs.

### Even Function

A function `f(x)` is considered an even function if, for every `x` in its domain, replacing `x` with `-x` results in the exact same output. Mathematically, this is expressed as `f(-x) = f(x)`. This means that the negative sign in the input does not alter the function's value. Graphically, even functions are always symmetrical about the y-axis.

**Examples of Even Functions:**

*   `f(x) = x^2`
*   `f(x) = |x|`
*   `f(x) = cos(x)`

### Odd Function

A function `f(x)` is considered an odd function if, for every `x` in its domain, replacing `x` with `-x` results in the negative of the original output. Mathematically, this is expressed as `f(-x) = -f(x)`. In this case, the negative sign in the input directly affects the function's value by reversing its sign. Graphically, odd functions exhibit symmetry about the origin.

**Examples of Odd Functions:**

*   `f(x) = x^3`
*   `f(x) = sin(x)`

## Conditions for Even and Odd Functions

To formally determine if a function is even or odd, we apply specific algebraic conditions:

*   **For an Even Function:** The condition `f(-x) = f(x)` must hold true for all `x` within the function's domain, `D(f)`. Equivalently, we can state that `f(-x) - f(x) = 0` for all `x` in `D(f)`.
*   **For an Odd Function:** The condition `f(-x) = -f(x)` must hold true for all `x` within the function's domain, `D(f)`. Equivalently, we can state that `f(-x) + f(x) = 0` for all `x` in `D(f)`.

## How to Check for Even and Odd Functions

The primary method to verify if a function is even or odd is the algebraic approach:

1.  **Substitute `-x`:** In the given function `f(x)`, replace every instance of `x` with `-x` to get `f(-x)`.
2.  **Simplify:** Simplify the expression for `f(-x)` as much as possible.
3.  **Compare:**
    *   If the simplified `f(-x)` is identical to the original `f(x)`, then the function is even.
    *   If the simplified `f(-x)` is the negative of the original `f(x)` (i.e., `f(-x) = -f(x)`), then the function is odd.
    *   If neither of these conditions is met, the function is neither even nor odd. Such functions are sometimes referred to as mixed functions.

## Examples of Checking Even and Odd Functions

Let's apply the algebraic method to a couple of common functions:

### Example 1: `f(x) = x^2`

1.  Substitute `-x`: `f(-x) = (-x)^2`
2.  Simplify: `f(-x) = x^2`
3.  Compare: Since `f(-x) = x^2` and `f(x) = x^2`, we have `f(-x) = f(x)`. Therefore, `f(x) = x^2` is an **even function**. This also applies to `x^4`, `x^6`, and other functions with even powers of `x`.

### Example 2: `f(x) = x^3`

1.  Substitute `-x`: `f(-x) = (-x)^3`
2.  Simplify: `f(-x) = -x^3`
3.  Compare: Since `f(-x) = -x^3` and `f(x) = x^3`, we have `f(-x) = -f(x)`. Therefore, `f(x) = x^3` is an **odd function**. This also applies to `x^5`, `x^7`, and other functions with odd powers of `x`.
