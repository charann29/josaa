# UI & Web Design using Adobe Illustrator CC by Udemy: Fee, Duration, How to Apply

This course offers a practical guide to mastering Adobe Illustrator CC for user interface (UI) and web design. It's designed to equip aspiring designers with the skills needed to create professional and responsive web layouts.

## Key Takeaways

*   Learn to use Adobe Illustrator CC specifically for UI and web design projects.
*   Understand essential concepts like wireframing, sitemaps, and responsive grid systems.
*   Gain practical experience in creating design elements, managing assets, and exporting for web.
*   The course is self-paced, delivered through video and text, and includes a certificate upon completion.

## Course Overview and Learning Format

This program focuses on teaching you how to leverage Adobe Illustrator CC to design websites and user interfaces. The learning experience is entirely self-study, allowing you to progress at your own speed. Content is delivered through a combination of video lectures and supporting text materials, all in English. Upon successful completion, a certificate is provided by Udemy, the platform offering the course.

## Fees and Certification

To enroll in the "UI & Web Design using Adobe Illustrator CC" course, the fee is ₹3,699. A certificate of completion is available once you finish the course content and meet any specified criteria. This certificate is issued directly by Udemy.

## Detailed Syllabus

The course covers a comprehensive range of topics, starting from foundational concepts and progressing to advanced design and export techniques. Here's a breakdown of what you'll learn:

### Getting Started with UI Design

*   An introduction to UI/UX design and the role of Illustrator.
*   Comparing Illustrator with other design tools like Photoshop, Sketch, and Adobe XD.
*   Setting up your Illustrator workspace and preferences for web design.
*   Understanding exercise files, completed project files, and handy cheat sheets.

### Planning Your Web Project

*   Distinguishing between sitemaps and wireframes.
*   Creating sitemaps and both low-fidelity (hand-drawn) and high-fidelity (Illustrator-based) wireframes.
*   Designing responsive wireframes for desktop, tablet, and mobile.

### Designing Pages and Elements

*   Determining appropriate screen sizes for various devices.
*   Implementing responsive 12-column grid systems.
*   Finding design inspiration and starting with Illustrator templates.
*   Integrating vector logos, managing colours using Adobe Color, and matching brand palettes.
*   Selecting and using web fonts from sources like Google Fonts and Typekit.

### Working with Images, Symbols, and Icons

*   Understanding free versus royalty-free images.
*   Techniques for adding, linking, cropping, masking, and washing out images in Illustrator.
*   Effective use of layers for organised design.
*   Creating and managing symbols and icons, including using resources like Adobe Market and Iconfinder.
*   Adjusting vector shapes, and drawing custom icons and logos with the Shape Builder and Pen tools.

### Responsive Design and Export

*   Developing tablet and mobile versions of your UI web design.
*   Exporting full-page web UI mockups.
*   Understanding SVG export and exporting images, logos, and icons for web and app design.
*   Preparing assets for Retina (HiDPI) and responsive image export.
*   Exporting CSS for developers directly from Illustrator.

### Career Insights

*   Learning the terminology of UI (User Interaction) design.
*   Guidance on how to secure your first job as a UI designer.
*   Suggestions for next steps to enhance your UI design career.

This course provides a robust foundation for anyone looking to enter the field of UI and web design using industry-standard tools.
