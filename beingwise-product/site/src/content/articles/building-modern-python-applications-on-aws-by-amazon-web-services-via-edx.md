# Building Modern Python Applications on AWS by Amazon Web Services via Edx: Fee, Duration, How to Apply

This program from Amazon Web Services (AWS) is designed to teach you how to build modern, scalable applications using Python within the AWS cloud environment. It focuses on serverless architectures, helping you create robust solutions without the overhead of managing traditional servers.

## Key Takeaways

*   Learn to develop serverless Python applications on AWS.
*   Understand how to use services like AWS Lambda, Amazon Cognito, and API Gateway.
*   Gain practical experience in designing cloud solutions and working with AWS CLI/API.
*   The course is self-paced, takes 1-4 hours per week, and lasts about six weeks.

## Course Overview

This certification program delves into creating scalable, serverless applications. You'll explore how to host services using virtual machines like Amazon EC2, and container services such as Amazon Elastic Kubernetes. A significant portion of the course covers APIs and their use with AWS Lambda and Amazon Cognito for authentication and authorization. You will also learn about integrating API Gateways with Lambda backends and configuring various Lambda functions. The program emphasizes building new, 'greenfield' serverless backends on AWS from scratch.

## What You Will Learn

By the end of this program, you will have a solid understanding of AWS technology and several Python applications. Key learning outcomes include:

*   Designing cloud architectures effectively.
*   Utilizing AWS Command Line Interface (CLI) and Application Programming Interfaces (APIs) to solve problems.
*   Gaining exposure to the Serverless Application Model (SAM).
*   Creating maps and models for cloud solutions.
*   Managing API Gateway controls.
*   Working with Amazon Cognito for user management.
*   Differentiating between synchronous and asynchronous responses in application design.
*   Building serverless workflows.
*   Becoming familiar with AWS X-Ray for application tracing and HTTP APIs.

## Who Is This Course For?

This program is ideal for computer science students or professionals who already possess a basic understanding of AWS concepts. This includes familiarity with the AWS global infrastructure, Identity and Access Management (IAM), Virtual Private Cloud (VPC), and other fundamental AWS terminology. It is particularly beneficial for those looking to build a promising career in cloud development and serverless architecture.

## Duration and Learning Effort

This is a self-paced, introductory-level course designed to be completed in approximately six weeks. The estimated learning effort is between one to four hours per week, allowing flexibility for learners to fit it into their schedules. The instruction medium is English, delivered through a mix of videos and text-based materials.

## Course Fee and Certification

The program offers two options: you can audit the course for a limited duration without any fee, or you can opt for a verified certificate by paying a fee of Rs. 12,323. To qualify for the certificate, you must upgrade your enrollment. While auditing, quizzes may not be graded, but for the verified certificate track, you will have unlimited attempts to improve your scores on assessments. Upon successful completion and payment, a certificate will be issued by Amazon Web Services, which can be a valuable addition to your resume and shared on professional platforms like LinkedIn.

This program provides a practical pathway to mastering modern Python application development on the AWS cloud, equipping you with skills highly sought after in today's tech landscape.
