# Ordered pair, Cartesian product of two sets

In mathematics, understanding how to combine elements from different sets in a structured way is fundamental. This page explores ordered pairs and the Cartesian product, key concepts in the broader study of sets, relations, and functions.

## Key takeaways

*   An ordered pair groups two elements where their sequence matters, like (a, b) is distinct from (b, a) unless a = b.
*   The Cartesian product of two sets, A and B, creates a new set containing all possible ordered pairs (a, b), where 'a' comes from A and 'b' comes from B.
*   This concept is crucial for Class 11 Mathematics and frequently appears in competitive exams like JEE Main and others such as BITSAT or SRMJEE.
*   The Cartesian product has specific properties, including non-commutativity and rules for cardinality and distribution over set operations.

## Understanding Ordered Pairs

An ordered pair is simply a collection of two items where the sequence in which they are listed is important. We typically write them as (a, b). For instance, (3, 5) is an ordered pair. The critical distinction is that (a, b) is generally not the same as (b, a). They are only considered equal if 'a' and 'b' themselves are identical.

Two ordered pairs, say (x, y) and (u, v), are equal if and only if their first elements are equal (x = u) AND their second elements are equal (y = v).

## What is a Cartesian Product?

The term "Cartesian" comes from the brilliant French mathematician and philosopher René Descartes. The Cartesian product of two non-empty sets, let's call them A and B, is a new set that includes every possible ordered pair you can form by taking the first element from set A and the second element from set B.

We denote this as A × B, read as "A cross B." Formally:

A × B = \{(a, b) : a ∈ A and b ∈ B\}

**Example:**

If Set A = \{1, 2\} and Set B = \{x, y\},

Then A × B = \{(1, x), (1, y), (2, x), (2, y)\}

### Cartesian Product involving an Empty Set

An empty set, denoted by φ, contains no elements. If you try to form a Cartesian product between any set (say, A) and an empty set, the result will always be an empty set. This is because you cannot form an ordered pair if one of the sets has no elements to contribute.

A × φ = φ

## Key Properties of the Cartesian Product

Understanding these properties helps in solving problems related to sets and relations.

### 1. Non-Commutative Nature

In general, A × B is not equal to B × A. This is because the order within the pairs matters. The pairs in A × B have their first element from A and second from B, while pairs in B × A have their first element from B and second from A. They are only identical if A and B are the same set.

### 2. Cardinality (Number of Elements)

If set A has 'm' elements (denoted as |A| = m) and set B has 'n' elements (denoted as |B| = n), then the Cartesian product A × B will have 'm × n' elements.

|A × B| = |A| ⋅ |B|

### 3. Distributive Properties

The Cartesian product distributes over set union and intersection, similar to how multiplication distributes over addition in arithmetic:

*   A × (B ∪ C) = (A × B) ∪ (A × C)
*   (A ∪ B) × C = (A × C) ∪ (B × C)
*   A × (B ∩ C) = (A × B) ∩ (A × C)
*   A × (B - C) = (A × B) - (A × C)

### 4. Relationships with Subsets

If A is a subset of B (A ⊆ B), then:

*   A × C is a subset of B × C (A × C ⊆ B × C)

These principles are fundamental in Class 11 Mathematics and are frequently tested in various engineering entrance examinations, including the JEE Main, as well as state-level exams.
