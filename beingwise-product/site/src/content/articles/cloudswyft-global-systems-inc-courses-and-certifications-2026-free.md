*   CloudSwyft provides online learning experiences focused on in-demand technology skills.
*   Many courses integrate with Microsoft Azure, a leading cloud platform.

*   Some courses offer pathways to official Microsoft certifications, enhancing career prospects.

## Advanced AI on Microsoft Azure: Deep Learning, Reinforcement Learning, and Applied AI

This micro-credential programme is tailored for undergraduate students at an intermediate level, focusing on advanced artificial intelligence concepts within the Microsoft Azure ecosystem. The curriculum covers deep learning, reinforcement learning, and practical AI applications.

Students engage in rigorous project work using cloud computing platforms provided by CloudSwyft and Microsoft. A key benefit of this course is the inclusion of an exam voucher for the Microsoft Azure AI Engineer Associate (AI-100) certification, allowing participants to pursue an industry-recognised credential. The programme spans 21 weeks and requires a fee for full access to materials and the completion certificate. 

Skills covered include:

*   Artificial Intelligence fundamentals
*   Data science principles
*   Python programming
*   Deep learning techniques
*   Machine learning concepts
*   Natural Language Processing (NLP)
*   Modelling methodologies

## Data Science Fundamentals on Microsoft Azure

Designed for beginners, this online course introduces the core concepts of data science and demonstrates how to leverage Microsoft Azure for creating AI solutions. It's an excellent starting point for individuals new to the field or professionals looking to transition their careers.

The programme is structured into four distinct courses, covering essential topics such as data analysis, research design in data science, experimental data analysis, modelling, and the application of Azure for AI. Learners will explore various Azure tools and services, including Azure Machine Learning, Azure SQL Database, Azure Synapse Analytics, and Azure Cognitive Services. While prior data science or Azure experience isn't mandatory, a basic understanding of mathematics and a commitment to self-paced study are beneficial. The course duration is 18 weeks and has a nominal programme fee.

Skills covered include:

*   Data science knowledge
*   Cloud computing concepts
