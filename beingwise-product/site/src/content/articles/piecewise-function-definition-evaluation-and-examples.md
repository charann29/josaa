# Piecewise Function: Definition, Evaluation & Examples

In mathematics, a piecewise function is a special type of function defined by multiple expressions, each applicable over a specific interval of its domain. These functions are incredibly useful for modelling real-world situations where a single mathematical rule doesn't quite capture the entire behaviour.

## Key Takeaways

*   A piecewise function uses different formulas for different parts of its input range.
*   It's a fundamental concept in higher mathematics, crucial for both board exams and competitive entrance tests like JEE Main.
*   Understanding piecewise functions helps in analysing scenarios where conditions change based on input values.
*   The absolute value function is a classic and simple example of a piecewise function.

## What is a Piecewise Function?

At its core, a function establishes a unique relationship where every input from a set (the domain) corresponds to exactly one output in another set (the codomain). A piecewise function extends this idea by defining different mathematical rules for different segments of its domain.

Think of it as having several mini-functions, each with its own specific range of inputs. When you provide an input value, the function checks which interval that value falls into and then applies the corresponding formula to calculate the output.

For instance, the absolute value function, $f(x) = |x|$, can be expressed as a piecewise function:

*   $f(x) = x$, when $x \ge 0$
*   $f(x) = -x$, when $x < 0$

Here, depending on whether $x$ is positive or negative, a different rule is applied to find $f(x)$.

## Why Piecewise Functions Matter

The concept of relations and functions is a cornerstone of mathematics, and piecewise functions are an important part of this foundation. They are frequently tested in various examinations, including board exams and significant competitive tests like the Joint Entrance Examination (JEE Main).

According to an analysis of past JEE Main papers, this topic has appeared consistently. For example, between 2013 and 2023, approximately six questions related to piecewise functions were asked, with notable appearances in 2014 (one question), 2020 (two questions), and 2021 (three questions). This highlights its importance for students aiming for top engineering colleges through exams like JEE Main, BITSAT, SRMJEE, and WBJEE.

## Examples of Piecewise Functions

Piecewise functions can take many forms, combining linear, quadratic, exponential, or other types of expressions. Here are a few common examples:

*   $f(x) = \begin{cases} x+2 & \text{ if } x<0 \\ x^2 & \text{ if } x \ge 0 \end{cases}$

    This function uses a linear rule for negative inputs and a quadratic rule for non-negative inputs.

*   $f(x) = \begin{cases} 1 & \text{ if } 0 \le x < 1 \\ 2 & \text{ if } 1 \le x < 2 \\ 3 & \text{ if } 2 \le x < 3 \end{cases}$

    This is an example of a step function, where the output is constant over specific intervals.

*   $f(x) = \begin{cases} 5 & \text{ if } x<-2 \\ 0 & \text{ if } -2 \le x \le 2 \\ -5 & \text{ if } x>2 \end{cases}$

    Here, the function has different constant values depending on the input's relation to -2 and 2.

## Graphing Piecewise Functions

Graphing a piecewise function involves plotting each individual expression over its specified interval. The resulting graph will be a collection of different curves or lines, joined or disconnected, at the points where the definition changes. It's crucial to pay attention to the boundaries of each interval, noting whether the points at these boundaries are included (closed circle) or excluded (open circle).

**Example:** Graph the piecewise defined function:

$f(x) = \begin{cases} -2x & \text{if } x < -2 \\ |x| & \text{if } -2 \le x \le 0 \\ 2 - x^2 & \text{if } x > 0 \end{cases}$

To graph this, you would:

1.  **For $x < -2$:** Plot the linear function $y = -2x$. Note that the point at $x = -2$ would be an open circle.
2.  **For $-2 \le x \le 0$:** Plot the absolute value function $y = |x|$. The points at $x = -2$ and $x = 0$ would be closed circles.
3.  **For $x > 0$:** Plot the quadratic function $y = 2 - x^2$. The point at $x = 0$ would be an open circle.

By combining these individual graphs, you get the complete visual representation of the piecewise function.
