*   Upon successful completion, participants receive a certificate from Udemy.

## What You Will Learn: The Syllabus

The course curriculum is meticulously designed to cover a broad spectrum of Python 3 topics, ensuring a holistic understanding. Here's a breakdown of the key modules:

### Foundations of Python

*   **Introduction and Environment Setup:** Get started with Python, understand its execution, install the interpreter, and set up the PyCharm IDE.
*   **First Python Program:** Write your first "Hello World" program, learn about variables, naming conventions, and adding comments to your code.
*   **Python Strings:** Explore string operations like splitting, counting, replacing, finding substrings, type conversion, and comparison.

### Control Flow and Functions

*   **Flow Control Statements:** Master comparison operators and conditional statements such as `if`, `else`, and `elif`, including nested `if` statements.
*   **Programming Iterative Loops:** Learn to use `while` and `for` loops, including `break` and `continue` statements, and `for` loops with an `else` clause.
*   **Python Functions:** Understand how to define and use functions, pass parameters, return values, and work with default, named, arbitrary, and keyword arguments, as well as anonymous functions.

### Object-Oriented Programming and Advanced Concepts

*   **Python Classes and Objects:** Dive into object-oriented programming by defining classes, creating objects, accessing attributes, defining methods, understanding class inheritance, and using the `del` keyword.
*   **Python Properties:** Learn about attributes, setter and getter methods, and the `@property` decorator.
*   **Polymorphism:** Grasp the concept of polymorphism, including its application with inheritance.
*   **Python Modules:** Understand how to create and import modules, import specific features, and explore Abstract Base Classes (ABCs).
*   **Protocols:** Learn about the Context Manager Protocol, Iterables, and Iterators.
*   **Decorators in Python:** Discover how to define and use decorators, including stacked and parameterized decorators, and class decorators.

### Data Structures

*   **Python Tuples:** Work with tuples, create new ones, iterate over them, and use related functions.
*   **Python Lists:** Explore lists, nested lists, list constructors, accessing, adding, concatenating, and removing elements.
*   **Python Sets:** Understand sets, their constructor, operations, and nesting.
*   **Python Dictionaries:** Learn about dictionaries, working with them, iterating over keys, values, and items, and checking key membership.

### Practical Application

*   **Python Projects for Beginners:** Apply your knowledge through hands-on projects such as a Dice Roll Game, a Number Guessing Game, and a Calculator, followed by a knowledge test.
