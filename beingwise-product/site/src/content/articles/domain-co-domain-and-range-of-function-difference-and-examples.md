# Domain, Co-domain and Range of Function: Difference and Examples

Understanding functions is fundamental in mathematics, forming the basis for countless real-world applications, from designing advanced vehicles to constructing towering skyscrapers. This concept, central to Class 11 Mathematics, is also vital for competitive exams like JEE Main and others.

## Key Takeaways

*   A function uniquely maps each element from its starting set (the domain) to exactly one element in its ending set (the co-domain).
*   The **domain** specifies all valid inputs for which the function is defined.
*   The **co-domain** is the complete set of all possible output values, including those that the function might not actually produce.
*   The **range** is a subset of the co-domain, consisting only of the actual output values generated by the function for its given domain.

## What is a Function?

In simple terms, a function is a special kind of relationship between two sets, let's call them Set A and Set B. For this relationship to be a function, every single element in Set A must be linked to one and only one element in Set B. We often write this as `f: A → B`, meaning 'f' is a function mapping from A to B.

For example, if a tree grows 20 cm each year, we can define a function `h(age) = age × 20`. Here, the tree's height (`h`) is a function of its `age`. If the age is 10 years, the height is `h(10) = 200 cm`.

It's important to note what isn't a function:

*   If an element in Set A isn't mapped to anything in Set B, it's not a function.
*   If an element in Set A is mapped to more than one element in Set B, it's also not a function.

## Understanding the Domain

The domain of a function is the collection of all valid input values for which the function produces a real and defined output. If a function `f` maps from Set A to Set B (`f: A → B`), then Set A itself is the domain of the function.

Let's look at some common examples of domains:

*   **Polynomial functions** (like linear, quadratic, or cubic functions): Their domain is all real numbers (R).
*   **Logarithmic functions** (e.g., `f(x) = log x`): The domain is `x > 0`, or `(0, ∞)`.
*   **Square root functions** (e.g., `f(x) = √x`): The domain includes all non-negative real numbers, denoted as `[0, ∞)`.
*   **Exponential functions**: Their domain is all real numbers (R).

## Understanding the Co-domain

If a function `f` is defined from Set A to Set B (`f: A → B`), then Set B is known as the co-domain of the function. The co-domain represents all *possible* output values, even if the function doesn't actually produce every single one of them. It's a larger or equal set that contains the range.

For instance, consider the function `f(x) = cos x`, where `f: R → R`. Here:

*   The domain is R (all real numbers).
*   The co-domain is also R (all real numbers).

However, the actual output values of `cos x` are always between -1 and 1. This leads us to the range.

## Understanding the Range

The range of a function is the set of all *actual* output values that the function produces when applied to every element in its domain. It's a subset of the co-domain.

Continuing with our example of `f(x) = cos x`, where `f: R → R`:

*   The co-domain is R.
*   The range, however, is `[-1, 1]`, because the cosine function's output never goes beyond these limits.

### Domain, Co-domain, and Range Comparison

Here's a quick comparison of these three crucial terms:

| Feature    | Domain                               | Co-domain                               | Range                                       |
| :--------- | :----------------------------------- | :-------------------------------------- | :------------------------------------------ |
| Definition | All possible input values            | All possible output values              | All actual output values                    |
| Relation   | The starting set A for `f: A → B`    | The ending set B for `f: A → B`         | A subset of the co-domain                   |
| Example    | For `f(x) = √x`, it's `[0, ∞)`       | For `f(x) = cos x` (`f: R → R`), it's R | For `f(x) = cos x` (`f: R → R`), it's `[-1, 1]` |

## Importance in Competitive Exams

The concepts of domain, co-domain, and range are frequently tested in engineering entrance exams. For example, in the JEE Main exam from 2013 to 2023, seven questions were specifically based on these ideas, including a notable four questions in 2023 alone. A solid grasp of these topics is therefore indispensable for students aiming for top scores.

Mastering these concepts will provide a strong foundation for advanced mathematical studies and problem-solving.
