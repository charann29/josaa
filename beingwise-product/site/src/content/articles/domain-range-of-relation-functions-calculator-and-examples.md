# Domain, Range of Relation: Functions, Calculator and Examples

In mathematics, understanding relations between sets is fundamental. The domain and range are crucial concepts that help us define and analyze these connections, showing how values behave within a given relation.

## Key takeaways

*   A relation links elements from one set to another, forming ordered pairs.
*   The domain of a relation is the collection of all first elements in these ordered pairs (the inputs).
*   The range of a relation is the collection of all second elements in these ordered pairs (the outputs).
*   These concepts are vital for Class 11 Mathematics and competitive exams like JEE Main.

## Understanding Relations

A relation R from a non-empty set A to a non-empty set B is essentially a selection of pairs from the Cartesian product A × B. This selection is based on a specific rule or description that connects an element from set A to an element from set B.

If an ordered pair (a, b) belongs to relation R, where 'a' is from set A and 'b' is from set B, we can write this as a R b. Here, 'b' is often referred to as the image of 'a'.

**Example:**

Let Set A = {1, 2} and Set B = {x, y}. A possible relation R could be: R = {(1, x), (2, y)}. This means 1 is related to x, and 2 is related to y.

## What is the Domain of a Relation?

The domain of a relation is the complete set of all the first components of the ordered pairs found within that relation. It represents all the possible input values that have at least one corresponding output value.

For instance, if a relation R is given as R = {(p, q), (r, s)}, then its domain would be {p, r}.

### How to Determine the Domain of a Function

When working with functions, finding the domain involves identifying all permissible input values (usually 'x'). Generally, the domain is the set of all real numbers, with certain restrictions:

*   **No division by zero:** If a function involves a fraction, the denominator cannot be zero. For example, in f(x) = 1/(x - 1), x cannot be 1.
*   **No negative numbers under even roots:** If a function includes an even root (like a square root), the expression inside the root must be non-negative. For instance, in f(x) = √(x - 3), x must be greater than or equal to 3.
*   **Explicitly defined intervals:** Sometimes, the function's domain is specified, such as f(x) = 3x + 4 for 2 < x < 12. In this case, x can only take values within that defined interval.

## What is the Range of a Relation?

The range of a relation is the complete set of all the second components of the ordered pairs within that relation. These are the output values that the relation produces.

Using our previous example, if R = {(p, q), (r, s)}, then its range would be {q, s}.

### How to Determine the Range of a Function

Finding the range of a function, often expressed as y = f(x), involves identifying all possible output values ('y').

*   **Consider the function's behavior:** Analyze how the function behaves for all values in its domain. What are the minimum and maximum output values? Are there any values the function can never reach?
*   **Algebraic manipulation:** Sometimes, you can rearrange the function to express 'x' in terms of 'y'. This can help identify restrictions on 'y'.
*   **Graphical analysis:** Sketching the graph of the function can visually reveal its range, showing the spread of y-values from the lowest to the highest point.

## Co-Domain of a Relation

The co-domain of a relation R from set A to set B is simply the entire set B itself. It's the set of all *possible* second elements, regardless of whether they are actually linked to an element in set A by the relation. The range is always a subset of the co-domain.

For example, if A = {1, 2} and B = {x, y, z}, and R = {(1, x), (2, y)}, then:

*   **Domain:** {1, 2}
*   **Range:** {x, y}
*   **Co-domain:** {x, y, z}

## Importance in Competitive Exams

Concepts of sets, relations, and functions are a significant part of the Class 11 Mathematics curriculum and are frequently tested in engineering entrance exams. According to an analysis of past papers, JEE Main has included several questions on this topic over the last decade, with a notable four questions appearing in 2023 alone. A solid grasp of domain and range is therefore essential for aspiring engineers.
