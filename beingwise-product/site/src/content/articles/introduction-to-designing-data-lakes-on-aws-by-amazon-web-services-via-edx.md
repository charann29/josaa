# Introduction to Designing Data Lakes on AWS by Amazon Web Services via Edx: Fee, Duration, How to Apply

This course offers an in-depth look at building and managing data lakes on Amazon Web Services (AWS), addressing the complexities that arise from vast and ever-growing data volumes. It's designed to equip IT professionals with the skills to create sustainable and scalable data solutions.

## Key Takeaways

*   Learn the core principles of data lakes and how to organize, refine, and manage large datasets efficiently.
*   Understand the distinction between data lakes and data warehouses, and apply best practices for AWS-based data lake architectures.
*   Gain practical knowledge of AWS services like Amazon S3, AWS Glue, Amazon Kinesis, and Amazon EMR for data movement and analysis.
*   Choose between a free audit track with limited access or a paid verified track that includes a certificate from Amazon Web Services.

## Course Structure and Effort

This program is structured as a 5-week course, requiring approximately 1 to 4 hours of study per week. It is delivered in English and follows a self-paced learning model, making it flexible for working professionals. The content is presented through a combination of video and text-based materials, ensuring a comprehensive learning experience.

## Who Should Attend?

This intermediate-level course is specifically tailored for IT Administrators and Architects who are responsible for designing, implementing, and maintaining data lakes. To benefit most from the curriculum, applicants should have 1 to 3 years of work experience in software development. A bachelor's degree or equivalent is also a prerequisite for enrollment.

## Learning Outcomes

Upon completing this program, you will be able to:

*   Articulate the fundamental concepts, characteristics, and structure of data lakes.
*   Differentiate between data lakes and data warehouses, understanding their respective applications.
*   Apply architectural protocols relevant to data lakes within the broader operational context.
*   Utilize core AWS services for data lake operations, including Amazon S3 for storage and AWS Glue Data Catalog.
*   Grasp the basics of machine learning and predictive analytics as applied within the AWS ecosystem.
*   Work with services like AWS Glue jobs, Amazon Kinesis Analytics, Amazon EMR, and AWS Lake Formation.

## Certification and Fees

The course offers two enrollment options: an audit track and a verified track. The audit track provides free, limited access to course materials but does not include a certificate. The verified track offers unlimited access to all course content and the opportunity to earn an official certificate from Amazon Web Services upon successful completion. The fee for the verified track, which includes the certificate, is $209.

To qualify for the certificate, participants in the verified track must complete weekly quizzes (which allow unlimited retakes and are ungraded) and pass a final course assessment. Only those who successfully clear this assessment will receive the certification.

This program offers a valuable opportunity to enhance your skills in a crucial area of modern data management, directly from the experts at Amazon Web Services.
