# Building Modern Java Applications on AWS by Amazon Web Services via Edx: Fee, Duration, How to Apply

For students and professionals keen on developing modern cloud-native applications, the "Building Modern Java Applications on AWS" online course offers a practical pathway. This introductory program, designed by Amazon Web Services and delivered through the Edx platform, focuses on equipping learners with the skills to build and manage scalable Java applications on the AWS cloud.

## Key takeaways

*   Learn to construct new, API-driven Java applications specifically for the AWS cloud environment.
*   Understand how to implement serverless workflows using AWS Step Functions and optimize APIs.
*   Gain practical experience with core AWS services like Lambda, API Gateway, and Cloud9.
*   Choose between a limited-access audit track or an unlimited-access verified track, with a certificate available upon upgrading.

## Course Overview and Learning Experience

This program delves into the specifics of developing modern, greenfield technology on AWS. It's structured to help you create business-ready projects and manage cloud applications effectively. The course is self-paced, allowing you to learn at your own speed, with an estimated commitment of 1-4 hours per week over a duration of six weeks.

The learning content is delivered through a blend of video lectures and text-based materials. You'll engage with assignments, projects, and video transcripts, guided by expert faculty. The curriculum covers essential topics such as cloud computing principles, AWS CLI, Cloud9, and various AWS service configurations.

## What You Will Learn

Upon completing this course, you will be able to:

*   Utilize the Serverless Application Model (SAM) to build professional serverless applications.
*   Design and create various files and models, enhancing your mapping skills for application development.
*   Solve real-world problems using the Amazon API Gateway.
*   Construct API applications securely with Amazon Cognito.
*   Implement AWS Lambda to create efficient serverless computing solutions.
*   Work effectively with Command Line Interfaces (CLIs) for AWS resource management.

## Eligibility Criteria

To get the most out of this course, participants should have foundational knowledge of AWS. This includes familiarity with:

*   The AWS global infrastructure, including Availability Zones and Regions.
*   High-level AWS Identity and Access Management (IAM) concepts and controlling access to AWS resources.
*   Basic understanding of Amazon EC2, VPC, and Amazon S3.

Essentially, a grasp of fundamental AWS terminology and services is crucial before enrolling.

## Tracks and Certification

The program offers two distinct learning tracks:

*   **Audit Track:** This track provides free, self-paced access to course materials for a limited duration.
*   **Verified Track:** For unlimited access to course content and the opportunity to earn a verified certificate, you must upgrade to this track. While the registration for the course itself is free, obtaining the certificate requires this upgrade.

The certificate is issued by Edx, in partnership with AWS, upon successful completion of the verified track requirements.

This course is an excellent opportunity to enhance your skills in building robust, scalable Java applications in the modern cloud environment provided by AWS.
