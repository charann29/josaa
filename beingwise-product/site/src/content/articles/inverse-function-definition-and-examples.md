# Inverse Function: Definition and Examples

An inverse function essentially reverses the action of another function. If a function takes an input and produces an output, its inverse takes that output and returns the original input.

## Key takeaways

*   An inverse function, denoted as $f^{-1}$, "undoes" what the original function $f$ does.
*   For an inverse function to exist, the original function must be both one-to-one (injective) and onto (surjective), making it a bijective function.
*   The composition of a function and its inverse always yields the original input: $(f \circ f^{-1})(x) = (f^{-1} \circ f)(x) = x$.
*   Understanding inverse functions is crucial for Class 11 Mathematics and competitive exams like JEE Main, SRMJEE, and BITSAT.

## What is an Inverse Function?

Imagine a function as a machine that processes an input to give a specific output. An inverse function is like a reverse machine that takes the output from the first machine and gives you back the exact original input. For instance, an exponential function takes a number and raises it to a power, while a logarithm (its inverse) takes that result and tells you what power was used.

Crucially, an inverse function $f^{-1}$ only exists if the original function $f$ satisfies two conditions:

1.  **One-to-one (Injective):** Every distinct input in the domain maps to a unique output in the codomain. No two different inputs produce the same output.
2.  **Onto (Surjective):** Every element in the codomain is an image of at least one element in the domain. There are no unused elements in the codomain.

When a function meets both these criteria, it is called a **bijective function**, and only bijective functions have inverse functions.

## Types of Inverse Functions

Inverse functions appear in various mathematical contexts, each with its own characteristics:

### Inverse Trigonometric Functions

These are often called "arc functions" because they determine the angle (or arc length) that corresponds to a given trigonometric ratio. There are six main inverse trigonometric functions:

*   arcsine (sin$^{-1}$)
*   arccosine (cos$^{-1}$)
*   arctangent (tan$^{-1}$)
*   arcsecant (sec$^{-1}$)
*   arccosecant (cosec$^{-1}$)
*   arccotangent (cot$^{-1}$)

### Inverse Rational Functions

A rational function is expressed as a ratio of two polynomials, $f(x) = P(x) / Q(x)$, where $Q(x)$ is not zero. To find its inverse, you typically follow these steps:

1.  Replace $f(x)$ with $y$.
2.  Swap $x$ and $y$ in the equation.
3.  Solve the new equation for $y$ in terms of $x$.
4.  Finally, replace $y$ with $f^{-1}(x)$ to represent the inverse function.

### Inverse Hyperbolic Functions

Similar to inverse trigonometric functions, inverse hyperbolic functions are the inverses of their corresponding hyperbolic functions. There are six primary inverse hyperbolic functions, such as inverse hyperbolic sine (arsinh) and inverse hyperbolic cosine (arccosh).

## Importance in Exams

The concept of inverse functions is a fundamental part of the "Relations and Functions" chapter in Class 11 Mathematics. It's not just important for school board exams but also for highly competitive entrance examinations. For example, analysis of past papers shows that the Joint Entrance Examination (JEE Main) has included questions on inverse functions. Between 2013 and 2023, a total of five questions related to this topic appeared in JEE Main, with notable occurrences in 2018, 2020, and three questions in 2021. This highlights its relevance for aspirants targeting exams like JEE Main, SRMJEE, BITSAT, WBJEE, and BCECE.

Mastering inverse functions will significantly strengthen your mathematical foundation for higher studies and competitive admissions.
