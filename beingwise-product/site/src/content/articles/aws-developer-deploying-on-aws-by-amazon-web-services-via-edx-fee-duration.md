# AWS Developer: Deploying on AWS by Amazon Web Services via Edx: Fee, Duration, How to Apply

This course, offered by Amazon Web Services through Edx, is the second in the AWS Developer Professional Series. It's designed for individuals looking to deepen their understanding of deploying applications on the AWS cloud, focusing on DevOps methodologies and practical application.

## Key Takeaways

*   An advanced-level program focusing on deploying applications using AWS services and DevOps tools.
*   Requires completion of the prerequisite course, "AWS Developer: Building on AWS," and prior software development experience.
*   Offers a hands-on learning approach with self-paced study, weekly discussions, and expert instructor support.
*   Provides an option to earn a verified certificate upon successful completion of a final assessment.

## Course Overview

"AWS Developer: Deploying on AWS" is structured as a six-week, self-paced online program. It delves into the practical aspects of using AWS tools like AWS Cloud9 to build and deploy applications. The curriculum is divided into modules, each containing lessons, exercises, quizzes, and opportunities for discussion with fellow learners and AWS training staff. Participants should plan to dedicate approximately three to four hours per week to the course material.

## Who Is This Course For?

This program is tailored for individuals who have already completed the first course in the series, "AWS Developer: Building on AWS." Additionally, it's ideal for those with one to three years of experience in software development, looking to advance their skills in cloud deployment and DevOps practices within the AWS ecosystem. The course assumes a foundational understanding of AWS and software development principles.

## What You Will Learn

The curriculum is designed to provide comprehensive knowledge and practical skills in deploying and managing applications on AWS. Key learning outcomes include:

*   **AWS Developer Tools:** Gain proficiency in using essential AWS developer tools such as AWS CloudFormation, AWS Elastic Beanstalk, AWS CodeDeploy, AWS CodeBuild, and AWS CodePipeline.
*   **Monitoring and Performance:** Learn how to effectively monitor development, production, and application performance using various AWS monitoring tools.
*   **Automation:** Master the techniques for automating production, development, and staging processes within the AWS environment.

## Course and Certificate Fees

The course offers an audit track, allowing learners to access most of the course material for free. For those seeking official recognition of their achievement, a verified certificate track is available for an upgrade fee. This certificate confirms successful completion of the course and can be shared on professional networks. To earn the verified certificate, learners must pass a final assessment with a minimum score of 65% on their first attempt.

## How to Apply

Applying for the "AWS Developer: Deploying on AWS" course is a straightforward online process. There is no traditional application form to fill out. Instead, prospective learners should follow these steps:

1.  **Visit the Course Page:** Navigate to the official course page on the Edx platform to review the program structure and details.
2.  **Enroll Now:** Locate and select the 'Enrol now' option on the course page.
3.  **Account Creation or Login:** Either create a new Edx account by providing your full name, email address, username, password, and country of residence, or log in using existing credentials (Google, Microsoft, Facebook, or Apple accounts).
4.  **Access Course Material:** Once logged in, you will gain immediate access to the course material via the audit track. If you wish to pursue the verified certificate, you can upgrade your enrollment at this stage.

This course offers a valuable opportunity to enhance your AWS deployment skills and contribute effectively to cloud-based development projects.
