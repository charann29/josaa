# Terms in Probability and Definitions of Probability

Probability is a fascinating branch of mathematics that helps us understand the likelihood of various outcomes in uncertain situations. It's a powerful tool for estimating chances and making predictions, with wide-ranging applications in real-world problem-solving.

## Key takeaways

*   Probability quantifies how likely an event is to happen, expressed as a value between 0 and 1.
*   A **random experiment** has multiple possible outcomes that cannot be predicted in advance.
*   The **sample space** lists all possible outcomes of a random experiment.
*   An **event** is a specific set of outcomes from a random experiment.

## What is Probability?

At its core, probability measures the chance of a particular event occurring. It's calculated as the ratio of the number of *favorable outcomes* (the outcomes we are interested in) to the *total number of possible outcomes*. This value always falls between 0 (meaning the event is impossible) and 1 (meaning the event is certain).

Mathematically, for an event E, the probability P(E) is given by:

P(E) = (Number of favorable outcomes for E) / (Total number of outcomes in the sample space)

## Random Experiment

A random experiment is a process or action that meets two specific criteria:

1.  **Multiple Outcomes:** It must have more than one possible result.
2.  **Unpredictable Outcome:** You cannot know with certainty which outcome will occur before the experiment is performed.

Even though you know all the potential results, you can't predict the specific one for any single trial. For instance, flipping a coin is a random experiment because it can land on either heads or tails (multiple outcomes), but you can't definitively say which it will be beforehand.

## Sample Space

The **sample space**, typically denoted by 'S', is the complete collection of all possible outcomes for a random experiment. Each individual outcome within the sample space is called a **sample point**.

Let's look at some examples:

*   **Rolling a single die:** If you roll a standard, unbiased six-sided die, the possible outcomes are 1, 2, 3, 4, 5, or 6. So, the sample space is S = {1, 2, 3, 4, 5, 6}.
*   **Tossing two coins simultaneously:** When two coins are tossed, the possible combinations of heads (H) and tails (T) are:
    *   Both coins show heads: (H, H)
    *   First coin heads, second coin tails: (H, T)
    *   First coin tails, second coin heads: (T, H)
    *   Both coins show tails: (T, T)
    
    Therefore, the sample space is S = {HH, HT, TH, TT}.

## Event

An **event** is simply a specific subset of outcomes from a random experiment's sample space. When the outcome of an experiment falls within the defined set of an event, we say that the event has occurred.

Consider the experiment of rolling a single die, where the sample space S = {1, 2, 3, 4, 5, 6}.

*   Let A be the event of rolling an even number. Then A = {2, 4, 6}.
*   Let B be the event of rolling a number greater than 4. Then B = {5, 6}.
*   Let C be the event of rolling a prime number. Then C = {2, 3, 5}.

Notice that events A, B, and C are all subsets of the sample space S.

### Types of Events

*   **Simple Event:** An event that contains only one sample point from the sample space. For example, if you toss a coin, getting 'Heads' is a simple event, as is getting 'Tails'.
*   **Mutually Exclusive Events:** Two or more events are mutually exclusive if they cannot happen at the same time. If one occurs, the others cannot. For example, when rolling a die, the event of rolling an even number and the event of rolling an odd number are mutually exclusive.
*   **Independent Events:** Events are independent if the occurrence or non-occurrence of one event has no impact on the occurrence or non-occurrence of another. For instance, flipping a coin twice: the outcome of the first flip does not affect the outcome of the second flip.

Understanding these fundamental terms is crucial for anyone delving into the world of probability, providing a solid foundation for more advanced concepts and real-world applications.
