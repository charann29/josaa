# Post Graduate Program in Artificial Intelligence and Machine Learning by Great Learning: Fee, Duration, How to Apply

For those looking to build a career in the rapidly expanding fields of Artificial Intelligence (AI) and Machine Learning (ML), Great Learning offers a Post Graduate Program designed to equip students with comprehensive skills. This program is structured to provide a deep understanding of core concepts and practical applications through a flexible learning format.

## Key Takeaways

*   The program is delivered entirely in English, utilizing a blend of self-study and virtual classroom sessions.
*   Learning materials are provided through video and text, with live classes typically scheduled on weekends.
*   Participants will earn a certificate upon successful completion, issued by Great Learning.
*   The total program fee is ₹3,75,000 plus GST, with various EMI options available.

## Program Structure and Learning Experience

This Post Graduate Program focuses on a hybrid learning model, combining the flexibility of self-paced study with the interactive engagement of virtual classrooms. All instruction is conducted in English, making it accessible to a wide audience. The course content is delivered through a combination of video lectures and textual materials, ensuring a comprehensive learning experience. To accommodate working professionals or those with other commitments, live virtual classes are typically held on weekends.

## Detailed Syllabus Overview

The curriculum is meticulously designed to cover foundational concepts, advanced machine learning techniques, and specialized topics in artificial intelligence. It's divided into three main sections:

### Foundations

This initial phase builds a strong base in essential tools and statistical methods:

*   **Python for AI & ML:** Covers Python basics, Jupyter Notebook, functions, packages (Pandas, NumPy, Matplotlib, Seaborn), and working with various data structures like arrays, vectors, and data frames.
*   **Applied Statistics:** Delves into descriptive and inferential statistics, probability (including conditional probability and distributions like Binomial, Poisson, and Normal), and hypothesis testing.

### Machine Learning

This section explores core machine learning algorithms and methodologies:

*   **Supervised Learning:** Includes topics such as multiple variable linear regression, logistic regression, K-Nearest Neighbors (K-NN), Naive Bayes classifiers, and Support Vector Machines (SVMs).
*   **Unsupervised Learning:** Focuses on K-means clustering, hierarchical clustering, high-dimensional clustering, and dimension reduction techniques like Principal Component Analysis (PCA).
*   **Ensemble Techniques:** Covers Decision Trees, Random Forests, Bagging, and Boosting methods.
*   **Featurization, Model Selection & Tuning:** Addresses feature engineering, model selection and performance measures, regularization, ML pipelines, and various cross-validation and search techniques (e.g., Grid Search CV, Randomized Search CV).
*   **Introduction to SQL:** Provides an understanding of database management systems (DBMS), ER diagrams, schema design, key constraints, normalization basics, joins, subqueries, analytic functions, and grouping/filtering.

### Artificial Intelligence

This advanced section dives into neural networks and specialized AI domains:

*   **Introduction to Neural Networks and Deep Learning:** Explores Gradient Descent, Perceptrons, Neural Networks, Batch Normalization, activation and loss functions, hyperparameter tuning, Deep Neural Networks, and frameworks like TensorFlow and Keras.
*   **Computer Vision:** Covers image data, Convolutional Neural Networks (CNNs), famous CNN architectures, transfer learning, object detection, semantic segmentation, instance segmentation, and metric learning (Siamese Networks, Triplet Loss).
*   **Natural Language Processing (NLP):** Introduces NLP concepts, text preprocessing, Bag of Words, TF-IDF, N-grams, Word2Vec, GLOVE, POS Tagging, Named Entity Recognition, and sequential models like Recurrent Neural Networks (RNNs) and Long Short-Term Memory (LSTM).

## Program Fees and Payment Options

The total program fee for the Post Graduate Program in Artificial Intelligence and Machine Learning is ₹3,75,000, plus Goods and Services Tax (GST). An admission fee of ₹30,000 (including GST) is payable upon enrollment.

Great Learning offers various Zero-Cost EMI plans to make the program more accessible. Here's a breakdown of the monthly payment amounts for different durations, with all figures including GST and any applicable processing charges:

| Installment No. | Payment (approx.) |
| :-------------- | :---------------- |
| 3 Months        | ₹1,43,400         |
| 6 Months        | ₹71,700           |
| 12 Months       | ₹35,850           |
| 18 Months       | ₹23,900           |
| 24 Months       | ₹20,251           |
| 36 Months       | ₹14,289           |

*   A scholarship of ₹15,000 may be available for selected candidates.
