# Building Containerized Applications on AWS by Amazon Web Services via Edx: Fee, Duration, How to Apply

This program introduces you to the world of container technology, a crucial skill in modern software development, specifically within the Amazon Web Services (AWS) ecosystem. It's designed to help you understand how to build and manage applications using containers on AWS.

## Key takeaways

*   Learn to differentiate between containers and virtual machines and their applications.
*   Gain practical experience with AWS services like Amazon Elastic Kubernetes Service (EKS) and Amazon Elastic Container Service (ECS) for container orchestration.
*   Understand how to monitor containerized applications using AWS CloudWatch Container Insights.
*   The course offers both a free audit track and a paid verified certificate option.

## Understanding Container Technology

Container technology provides a way to package applications with all their dependencies, ensuring they run consistently across different environments. Popular container tools include Docker and Kubernetes, which are powerful but can be complex to manage. This course tackles these challenges by showing you how to leverage AWS services to simplify the orchestration and deployment of containerized applications.

## What You Will Learn

By the end of this program, you will have a solid understanding of:

*   The fundamental concepts of containers and how they differ from traditional virtual machines.
*   How to effectively use Amazon EKS and Amazon ECS to manage and scale your containerized applications.
*   Techniques for enabling communication between different parts of your application using AWS App Mesh.
*   Methods for collecting performance metrics and logs from your containerized applications with AWS CloudWatch Container Insights.
*   Practical application of Docker containers in an AWS environment.

## Course Structure and Learning Experience

This is an intermediate-level, self-paced course delivered through pre-recorded video lectures and text-based materials. It requires approximately 2-4 hours of study per week and is designed to be completed in about five weeks. The program emphasizes hands-on learning through lab exercises and practical assignments, guided by expert AWS educators. You'll also have access to discussion groups and weekly quizzes to reinforce your learning.

### Program Highlights

*   **Learning Mode:** Self-paced, video, and text-based.
*   **Duration:** Approximately five weeks, 2-4 hours per week.
*   **Level:** Intermediate.
*   **Content:** AWS technical demonstrations, practical lab experience, weekly quizzes.
*   **Instructors:** Trained AWS educators.

## Eligibility and Prerequisites

To get the most out of this course, it is recommended that you have prior experience in application development and familiarity with the AWS Management Console. Knowledge of the AWS Developer Professional Series, including building, deploying, and optimizing on AWS, would also be beneficial.

## Fees and Certification

The course is available in two tracks:

*   **Free Audit Track:** This track provides access to the course content for learning purposes without a certificate.
*   **Verified Certificate Track:** For a fee of ₹7,414, you gain unlimited access to the course materials and, upon successful completion of a final assessment, you will receive a shareable digital certificate from Amazon Web Services. This certificate can be a valuable addition to your professional profile.
