# Post Graduate Program in External Aerodynamic Simulations by Skill Lync: Fee, Duration, How to Apply

This program is designed for aspiring engineers looking to specialize in external aerodynamic simulations, a crucial area in automotive, aerospace, and other industries. It offers a deep dive into advanced simulation techniques and software, preparing students for specialized roles.

## Key Takeaways

*   **Comprehensive Curriculum:** The program covers essential tools like ANSA for advanced meshing and ANSYS Fluent for CFD, alongside MATLAB for engineering applications.
*   **Practical Focus:** Emphasizes hands-on learning through projects and real-world case studies, ensuring practical skill development.
*   **Flexible Learning:** Delivered through self-study, video, and text-based modules, allowing students to learn at their own pace.
*   **Industry-Relevant Skills:** Focuses on skills highly sought after in the external aerodynamics and computational fluid dynamics (CFD) fields.

## Program Structure and Duration

This Post Graduate Program is a self-paced, video and text-based learning experience, conducted entirely in English. It is structured into three main courses, each focusing on a distinct aspect of aerodynamic simulations and engineering tools:

1.  **Advanced CFD Meshing using ANSA:** This course is spread over five weeks and introduces participants to the ANSA graphical user interface (GUI), geometric tools, topology cleanup, and various meshing techniques. It progresses from 2D and 3D meshing to complex CFD meshing for models like a BMW M6 inside a wind tunnel and surface wrapping for automotive assemblies.
2.  **Introduction to GUI based CFD using ANSYS Fluent:** This is an intensive nine-week module covering the fundamentals of CFD, governing equations, and numerical discretization. It delves into setting up steady-state simulations, exploring meshing strategies (including Y+ estimation and inflation layers), external aerodynamics (virtual wind tunnels, drag, and downforce calculations), Conjugate Heat Transfer, Discrete Phase Modelling, and an introduction to User Defined Functions and basic reacting flows.
3.  **MATLAB for Mechanical Engineers:** This introductory module for Mechanical Engineers covers MATLAB syntax, commands, and their application in various scenarios, including data manipulation.

## Program Fees

The total fee for the Post Graduate Program in External Aerodynamic Simulations is ₹275,000. For those preferring installment options, an EMI plan is available at ₹14,375 per month. The program fee covers all course materials and access to the learning platform.

## Certification

Upon successful completion of the program, participants will receive a certificate directly from Skill Lync. This certification acknowledges the skills and knowledge gained in advanced CFD meshing, GUI-based CFD using ANSYS Fluent, and MATLAB for mechanical engineering applications.
