# Question: 51% of a whole number is 714. What is 25% of that number?

This type of percentage-based problem is common in many competitive exams and aptitude tests. Understanding how to approach these questions systematically can significantly improve your speed and accuracy. Let's break down the solution.

## Key Takeaways

*   Percentage problems often require you to first find the 'whole' number before calculating a different percentage of it.
*   Representing the unknown number with a variable (like 'x') simplifies the setup of the equation.
*   Converting percentages to fractions (e.g., 51% as 51/100) is a standard method for calculations.
*   Careful algebraic manipulation is crucial to avoid errors in solving for the unknown.

## Understanding the Problem Statement

The question presents two pieces of information: a specific percentage (51%) of an unknown whole number results in a known value (714), and then asks us to find a different percentage (25%) of that same unknown number. The first step is always to use the given information to find the original whole number.

## Step-by-Step Solution

To solve this, we'll follow a clear, two-stage process:

### Stage 1: Finding the Original Number

Let's assume the unknown whole number is 'x'.

The problem states that 51% of 'x' is equal to 714. We can write this as an equation:

51% of x = 714

To perform calculations, we convert the percentage to a fraction or decimal. 51% is equivalent to 51/100.

So, the equation becomes:

(51/100) * x = 714

To find 'x', we need to isolate it. We can do this by multiplying both sides of the equation by 100 and dividing by 51:

x = 714 * (100 / 51)

Now, perform the division:

714 / 51 = 14

So, x = 14 * 100

x = 1400

Thus, the original whole number is 1400.

### Stage 2: Calculating 25% of the Original Number

Now that we know the original number is 1400, the second part of the question asks for 25% of this number.

Similar to before, convert 25% to a fraction (25/100) or a decimal (0.25).

25% of 1400 = (25/100) * 1400

We can simplify this calculation:

25/100 is equal to 1/4.

So, (1/4) * 1400 = 1400 / 4

1400 / 4 = 350

Therefore, 25% of the number is 350.

## Conclusion

The final answer is 350. By systematically breaking down the problem into finding the whole number first and then calculating the desired percentage, these types of questions become straightforward. This method is applicable across various quantitative aptitude sections in exams.
