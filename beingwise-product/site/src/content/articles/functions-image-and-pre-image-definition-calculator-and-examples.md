# Functions, Image and Pre-image: Definition, Calculator and Examples

In mathematics, functions, along with their images and pre-images, are fundamental concepts that help us understand how different values are connected through a specific rule or mapping. The image refers to the output generated by a function, while the pre-image represents the input values that lead to those outputs.

## Key Takeaways

*   A function is a special relationship where each input has exactly one output.
*   The **image** of a function is the set of all possible output values (the range).
*   The **pre-image** of a function refers to the input values that produce a particular output.
*   Understanding these concepts is crucial for topics like domain, range, and various types of function mappings in algebra and set theory.

## What is a Function in Mathematics?

A function is a specific type of relationship between two sets, let's call them Set A and Set B. For a relationship to be considered a function, every single element in Set A (the input set) must be paired with one and only one element in Set B (the output set).

Formally, if we have a relation from Set A to Set B, it is a function if each element in Set A corresponds to exactly one image in Set B. We often write this as `f: A → B`, which means 'f is a mapping from A to B'.

**Example:**

Consider the function `f(x) = 2x + 1`. If our input set A is `{1, 2, 3}`, then the function maps these inputs as follows:

*   `f(1) = 3`
*   `f(2) = 5`
*   `f(3) = 7`

Here, each input from Set A has a unique output in the resulting set `{3, 5, 7}`.

## Understanding Domain, Range, and Mapping

These terms help clarify the scope and behavior of a function:

*   **Domain:** This is the complete collection of all possible input values for which the function is defined. These inputs are the pre-images. For instance, in the function `f(x) = x^2`, the domain can be all real numbers (`ℝ`).
*   **Range:** This is the collection of all possible output values that the function can produce. These outputs are the images. For `f(x) = x^2`, the range includes all non-negative real numbers, written as `[0, ∞)`.
*   **Mapping:** This describes the specific pairing between each input from the domain and its corresponding output in the range. Mapping diagrams use arrows to visually represent these connections, making it clear how pre-images lead to their respective images.

### Visualizing Functions and Non-Functions

When looking at diagrams, it's easy to spot what makes a relationship a function or not:

*   **A valid function:** Every element in the input set (Domain) points to exactly one element in the output set (Codomain).
*   **Not a function (Missing input):** If an element in the input set is not connected to any element in the output set, it's not a function.
*   **Not a function (Multiple outputs):** If an element in the input set is connected to more than one element in the output set, it's not a function.

## Image of a Function

The image of a function refers to the entire collection of output values that the function generates when applied to its domain. It is essentially the range of the function.

### Definition of Image

Given a function `f: A → B` (meaning `f` maps elements from set A to set B), and a subset `X` within `A`, the image of `X` under `f` is defined as:

`f(X) = { f(x) | x ∈ X }`

If we consider the entire domain `A`, the image of the function `f` (which is also its range) is:

`Image(f) = f(A) = { f(a) | a ∈ A }`

**Important Considerations:**

1.  Always specify whether you are referring to the image of a single element or the image of an entire subset.
2.  `f(x)` represents a single output element, while `f(C)` (where `C` is a subset of the domain) represents a subset of the codomain.
3.  If an element `y` is part of `f(C)`, it means `y` belongs to Set B, and there must be at least one element `x` in `C` such that `f(x) = y`.

## Pre-image of a Function

The pre-image of a function refers to the input values (from the domain) that map to a specific output value or a set of output values in the codomain.

### Definition of Pre-image

For a function `f: A → B` and a subset `Y` within `B` (the codomain), the pre-image of `Y` under `f` is defined as:

`f⁻¹(Y) = { x ∈ A | f(x) ∈ Y }`

In simpler terms, `f⁻¹(Y)` gathers all the elements from Set A that, when put through the function `f`, produce an output found within the set `Y`.

**Example:**

Let `f(x) = x^2`. If we want to find the pre-image of the value `9` (i.e., `Y = {9}`), we look for all `x` such that `x^2 = 9`. In this case, `x` could be `3` or `-3`. So, `f⁻¹({9}) = {3, -3}`.

## Image vs. Pre-image in Functions

The key distinction between image and pre-image lies in their direction within the function's mapping:

| Feature     | Image (Output)                                   | Pre-image (Input)                                    |
| :---------- | :----------------------------------------------- | :--------------------------------------------------- |
| **Direction** | From domain to codomain                          | From codomain to domain                              |
| **Represents**| The result or output of the function             | The original input(s) that lead to a specific output |
| **Notation**  | `f(x)` for an element, `f(X)` for a subset       | `f⁻¹(y)` for an element, `f⁻¹(Y)` for a subset       |
| **Location**  | Elements found in the codomain (or range)        | Elements found in the domain                         |

Understanding these foundational concepts is essential for higher-level mathematics and various practical applications where relationships between quantities are analyzed.
