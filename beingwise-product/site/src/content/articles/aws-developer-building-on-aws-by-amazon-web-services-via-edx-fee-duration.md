# AWS Developer: Building on AWS by Amazon Web Services via Edx: Fee, Duration, How to Apply

For aspiring developers looking to master cloud computing, the AWS Developer: Building on AWS online course offers a structured path to building and deploying applications on the Amazon Web Services (AWS) platform. This program, designed for self-paced learning, equips you with practical skills to navigate the vast AWS ecosystem.

## Key takeaways

*   This is a six-week, advanced-level program from AWS itself, delivered in English through video lectures and text.
*   It requires approximately 3-4 hours of study per week and is accessible for free through its audit track.
*   Participants will gain hands-on experience building web applications using core AWS services like Lambda and EC2.
*   A verified digital certificate is available upon successful completion of the paid track, confirming your AWS development skills.

## Course Overview

This six-week online course is tailored to teach you the essentials of developing and running applications within the AWS environment. You'll begin by setting up your AWS account and installing the necessary Software Development Kits (SDKs). The curriculum covers fundamental AWS concepts such as regions, availability zones, security groups, and Virtual Private Clouds (VPCs).

A significant portion of the course is dedicated to practical application. You'll engage in building a web application using key AWS compute services like AWS Lambda and Amazon EC2. Additionally, the program introduces you to various other AWS offerings, including Amazon Rekognition, AWS SNS, and Amazon SQS. Learning is reinforced through video lectures, reading materials, quizzes, practical exercises, and class discussions.

## What You Will Learn

Upon completing the AWS Developer: Building on AWS program, you will develop proficiency in several critical areas:

*   Understanding fundamental AWS concepts such as regions, Availability Zones, and VPCs.
*   The process of creating and effectively managing an AWS account.
*   Utilizing AWS compute services both programmatically and through the AWS console.
*   Working with Amazon AI services, including text-to-speech and image recognition applications.
*   Installing and effectively using the AWS SDKs for development.
*   Interacting with AWS APIs via the AWS Cloud9 Integrated Development Environment (IDE), AWS Command Line Interface (CLI), and AWS SDKs.
*   Employing AWS managed services like Amazon RDS.
*   Designing applications to be highly available, distributed, and durable within the AWS cloud.

## Eligibility and Certification

To enroll in this AWS Developer training, you should ideally possess 1-3 years of prior software development experience. Additionally, a credit card is required to set up your AWS account, which is a necessary step for the hands-on components of the course.

While the course content is freely accessible through the audit path, obtaining a shareable digital certificate requires pursuing the verified certificate track. This verified certificate confirms your successful completion of the program and can be a valuable addition to your professional profile.

## How to Apply

The application process for the AWS Developer: Building on AWS course is straightforward and entirely online:

1.  Visit the official course page on the Edx platform.
2.  Locate and click the 'Enroll now' button, typically found near the top of the page.
3.  A signup form will appear. You can sign up by linking an existing account (such as Apple, Google, Facebook, or Microsoft) or by entering your details (username, email ID, full name, country, and a new password) to create a new account.
4.  Once registered, you will be enrolled in the program. You then have the option to begin learning for free via the audit path or to purchase the verified track to be eligible for the certificate.

This program offers a valuable opportunity to enhance your cloud development skills directly from AWS, preparing you for a career in cloud computing.
