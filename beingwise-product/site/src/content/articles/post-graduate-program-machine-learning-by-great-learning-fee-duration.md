# Post Graduate Program Machine Learning by Great Learning: Fee, Duration, How to Apply

For students and professionals keen to enter the dynamic field of Machine Learning, the Post Graduate Program in Machine Learning offers a comprehensive pathway. This program is designed to equip learners with essential skills and knowledge through a structured curriculum.

## Key takeaways

*   The program is delivered entirely in English, combining self-study with virtual classroom sessions and video/text-based content.
*   The total program fee is ₹1,50,000 plus GST, with various EMI options available.
*   The curriculum covers foundational topics like Python and statistics, core machine learning concepts, and additional modules such as time series forecasting and model deployment.
*   Upon successful completion, participants receive a certificate directly from Great Learning.

## Program Structure and Delivery

This Post Graduate Program in Machine Learning is designed for flexibility and comprehensive learning. The instruction medium is English, and the learning experience combines self-paced study with interactive virtual classroom sessions. Content is delivered through a mix of video and text-based materials, allowing students to learn at their own convenience while also benefiting from live interactions.

The total fee for the Post Graduate Program in Machine Learning is ₹1,50,000, in addition to applicable Goods and Services Tax (GST). To make the program accessible, several Equated Monthly Installment (EMI) plans are available, catering to different financial needs. ## Detailed Syllabus

The program curriculum is structured to provide a robust understanding of Machine Learning, starting from fundamental concepts and progressing to advanced topics and practical application. It is divided into several key sections:

### Foundations

*   **Python for AI & ML:** Essential programming skills using Python, tailored for Artificial Intelligence and Machine Learning applications.
*   **Applied Statistics:** Core statistical concepts vital for understanding and interpreting data in ML contexts.

### Machine Learning

*   **Supervised Learning:** In-depth study of algorithms where models learn from labeled data.
*   **Unsupervised Learning:** Exploration of techniques for finding patterns in unlabeled data.
*   **Ensemble Techniques:** Methods that combine multiple models to improve predictive performance.
*   **Featurization, Model Selection & Tuning:** Strategies for preparing data, choosing appropriate models, and optimizing their performance.
*   **Introduction to SQL module:** Basic SQL skills for data manipulation and retrieval, crucial for data scientists.

### Additional Modules

*   **EDA (Exploratory Data Analysis):** Techniques for analyzing data sets to summarize their main characteristics, often with visual methods.
*   **Time Series Forecasting:** Methods for predicting future values based on historical time-ordered data.
*   **Model Deployment:** The process of integrating a machine learning model into an existing production environment.

### Capstone Project

The program culminates in a Capstone Project, allowing participants to apply their acquired knowledge and skills to solve a real-world problem, thereby solidifying their understanding and building a portfolio piece.

Upon successful completion of all program requirements, participants will receive a certificate from Great Learning, acknowledging their achievement and newly acquired expertise in Machine Learning.

This program offers a well-rounded opportunity for individuals looking to build a career in the rapidly expanding field of Machine Learning.
