# Post Graduate Program in Computational Fluid Dynamics: Fee, Duration, How to Apply

Computational Fluid Dynamics (CFD) is a vital field in engineering, enabling the simulation of fluid flows and heat transfer. This program offers an in-depth exploration of CFD, equipping participants with practical skills using industry-standard software.

## Key Takeaways

*   The program is delivered in English, primarily through self-study with video and text-based materials.
*   It covers advanced CFD meshing, GUI-based CFD using ANSYS Fluent, and an introduction to CFD using MATLAB and OpenFOAM.
*   The total program fee is Rs. 2,75,000, with an EMI option available at Rs. 14,375 per month.
*   Upon successful completion, participants receive a certificate from the program provider.

## Program Structure and Learning Approach

This Post Graduate Program in Computational Fluid Dynamics is structured for self-paced learning, providing flexibility for participants. The instructional content is entirely in English, delivered through a combination of video lectures and supporting text materials. This format allows learners to progress at their own speed, revisiting complex topics as needed.

## Detailed Curriculum Overview

The program is divided into three main courses, each focusing on different aspects and tools within Computational Fluid Dynamics. Here's a look at what each course covers:

### Course 1: Advanced CFD Meshing using ANSA

This initial course delves into the critical skill of creating high-quality meshes for CFD simulations using ANSA software. It's broken down into weekly modules:

*   **Week 1: Introduction to ANSA GUI and Tools** - Learn the basics of the ANSA interface, geometric tools, and topology cleanup.
*   **Week 2: 2D (Surface) Meshing to Pressure Valve** - Focus on creating 2D surface meshes, PID creation, and basic topology cleanup for components like a pressure valve.
*   **Week 3: 3D (Volume) Meshing to Turbocharger** - Move to 3D volumetric meshing, including geometry cleanup for defining volumes and ensuring mesh quality for complex parts such as a turbocharger.
*   **Week 4: CFD Meshing to BMW M6 Model inside Wind Tunnel** - Tackle advanced topology cleanup, variable surface meshing, addressing quality failures, and creating wind tunnel meshes for automotive models.
*   **Week 5: Surface Wrap to an Automotive Assembly** - Master geometry cleanup for surface wrapping and merging multiple models for an assembly.

### Course 2: Introduction to GUI-based CFD using ANSYS Fluent

This course focuses on practical CFD simulations using the widely-used ANSYS Fluent software. Key topics include:

*   **Week 1: Introduction to CFD** - Fundamentals of CFD, including governing equations, numerical discretization, fluid solvers, boundary conditions, and post-processing.
*   **Week 2: Simulating Laminar and Turbulent Flows in ANSYS Fluent** - Hands-on experience with geometry creation, meshing, setting up boundary conditions, and post-processing simulations.
*   **Week 3: Performing Steady State Simulations** - Learn to set up steady-state simulations, check for convergence, and create runtime animations. Includes projects like HVAC simulation, parametric studies on gate valves, and cyclone separator performance characterization.
*   **Week 4: Exploring Meshing Strategies** - Advanced meshing techniques such as local refinement, understanding Y-plus, inflation layers, and mesh dependence tests.
*   **Week 5: External Aerodynamics** - Setting up virtual wind tunnels, analyzing vortices, calculating downforce and drag on vehicles, and Y+ estimation.
*   **Week 6: Conjugate Heat Transfer** - Extracting solid and fluid volumes, creating shared topologies for conformal meshes, setting up volumetric heat sources, and visualizing heat transfer coefficients.
*   **Week 7: Discrete Phase Modelling** - Explore different discrete phase boundary conditions, methods of tracking particles, turbulence intensity, and vortex core visualization.
*   **Week 8: Introduction to User Defined Functions** - A brief overview of UDFs.
*   **Week 9: Basic Reacting Flows** - An introduction to simulating reacting flows.

### Course 3: Introduction to CFD using MATLAB and OpenFOAM

The final course introduces participants to CFD using programming environments like MATLAB and the open-source OpenFOAM platform.

*   **Week 1: What is Computational Fluid Dynamics?** - A deeper dive into CFD, its necessity, advantages, modeling process, and a detailed understanding of the Navier-Stokes equations, including continuity, momentum, energy equations, and the significance of the Reynolds number.
*   **Week 2: Mathematics and Fluid Dynamics Essentials** - This segment covers the essential mathematical and fluid dynamics concepts required for CFD.

## Program Fees and Payment Options

The total fee for the Post Graduate Program in Computational Fluid Dynamics is **Rs. 2,75,000**. For those who prefer to pay in installments, an Equated Monthly Installment (EMI) option is available, priced at **Rs. 14,375 per month**.

## Certification

Upon successful completion of all program requirements, participants will receive a certificate directly from the program provider. This certification validates the skills and knowledge acquired in advanced computational fluid dynamics.

This program provides a comprehensive pathway for individuals looking to build expertise in Computational Fluid Dynamics, leveraging both commercial and open-source tools.
