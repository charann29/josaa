# Question: 51% of a whole number is 714. What is 25% of that number?

This type of percentage-based problem is common in many competitive exams and aptitude tests. It tests your basic understanding of how to work with percentages and solve for an unknown value. Let's break down the solution step-by-step.

## Key Takeaways

*   Understanding percentages is fundamental for many quantitative aptitude questions.
*   To find the original number when a percentage is given, set up a simple algebraic equation.
*   Once the original number is found, calculating any other percentage of it becomes straightforward.
*   Practice with such problems helps build speed and accuracy for exams.

## Understanding the Problem

The question presents us with a scenario where 51% of an unknown whole number is known to be 714. Our goal is to first figure out what that whole number is, and then calculate 25% of that number. This involves two distinct steps of percentage calculation.

## Step 1: Finding the Original Number

Let's assume the unknown whole number is 'X'.

According to the problem statement:

51% of X = 714

To express 51% as a fraction or decimal, we write it as 51/100.

So, the equation becomes:

(51/100) * X = 714

To find X, we need to isolate it. We can do this by multiplying both sides of the equation by (100/51):

X = 714 * (100/51)

Now, let's perform the division. 714 divided by 51 is 14.

714 / 51 = 14

So, X = 14 * 100

X = 1400

The original whole number is 1400.

## Step 2: Calculating 25% of the Original Number

Now that we know the original number is 1400, the next step is to find 25% of it.

25% of 1400 = (25/100) * 1400

We can simplify this calculation:

(1/4) * 1400

1400 / 4 = 350

So, 25% of the original number is 350.

## Checking the Options

The options provided were:

*   Option 1: 350
*   Option 2: 450
*   Option 3: 550
*   Option 4: 250

Our calculated answer, 350, matches Option 1.
